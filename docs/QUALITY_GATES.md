# Quality Gates

## Gate 0 — Repository integrity

- [ ] Doğru package manager kullanılıyor.
- [ ] Lockfile korunuyor.
- [ ] Mevcut stack gereksiz değiştirilmedi.
- [ ] Secrets veya binary junk eklenmedi.

## Gate 1 — Brief completeness

- [ ] Yedi zorunlu route mevcut.
- [ ] Header ve footer mevcut.
- [ ] Hizmet dropdown mevcut.
- [ ] Mobile hamburger ve accordion çalışıyor.
- [ ] Tüm zorunlu içerik bölümleri mevcut.
- [ ] SEO metadata mevcut.
- [ ] Görseller ve alt text mevcut.
- [ ] Contact form graceful davranıyor.

## Gate 2 — Technical health

- [ ] Install başarılı.
- [ ] Build başarılı.
- [ ] Lint başarılı.
- [ ] Typecheck başarılı.
- [ ] Test başarılı.
- [ ] Browser console error yok.
- [ ] Broken request yok.

## Gate 3 — Functional UX

- [ ] Tüm navigasyon linkleri çalışıyor.
- [ ] Dropdown/hamburger/accordion çalışıyor.
- [ ] CTA'lar doğru hedefe gidiyor.
- [ ] Form etkileşimi çalışıyor.
- [ ] Klavye focus görünür.
- [ ] Hit area yeterli.

## Gate 4 — Responsive

- [ ] 1440×1000
- [ ] 1024×1366
- [ ] 390×844
- [ ] 360×800
- [ ] Yatay overflow yok.
- [ ] Text clipping yok.
- [ ] Görsel crop hatası yok.
- [ ] Mobile menu viewport dışına taşmıyor.

## Gate 5 — Visual quality

- [ ] Premium ve kurumsal ilk izlenim.
- [ ] Generic dashboard/template hissi yok.
- [ ] Hero güçlü ve özgün.
- [ ] Typography hiyerarşisi net.
- [ ] Spacing ritmi tutarlı.
- [ ] Renk aksanları kontrollü.
- [ ] Kart monotonluğu yok.
- [ ] Görseller kaliteli ve doğru entegre.
- [ ] Critical/high visual issue yok.
- [ ] Ortalama skor ≥ 82/100.

## Gate 6 — Delivery evidence

- [ ] Test raporu kaydedildi.
- [ ] Screenshot'lar kaydedildi.
- [ ] Visual review raporları kaydedildi.
- [ ] `CHANGELOG_AI.md` güncel.
- [ ] `ACTIVE_TASK.md` PASS.
- [ ] Bilinen limitler dürüstçe yazıldı.
